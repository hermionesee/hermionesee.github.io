Murphy's Dive

Introduction

Murphy's Dive is a dungeon crawler action shooter where players navigate through challenging levels to find and defeat the final boss. Collaborated as a team of 8 to create a 2D game in C++ and Lua, inspired by games “Enter the Gungeon” and “The Binding of Issac” using our team’s custom game engine

  • Controls

    ‣ Basic WASD keys for movement

    ‣ Mouse for Aiming

    ‣ Left Click for Shooting, Right Click for Ultimate

  • Unique Selling Point

    Gimmicky Weapons

Murphy's Custom Engine


Using C++, I implemented the game’s collision logic with Axis-Aligned Bounding Box detection applied throughout the game world, developed an animation system leveraging numerically labeled sprite sheet frames for efficient state management, and created a cutscene manager to handle scripted story events for improved narrative delivery. Additionally, I extended the existing Collectible system to support Lua-based power-up mechanics, enhancing gameplay variety — all built within my group’s custom C++ engine.

The engine is created using IMGui.
